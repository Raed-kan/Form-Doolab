# ربط الاستبيان بـ Google Sheet — خطوة بخطوة

الهدف: كل عميل يعبّي الاستبيان → يضاف صف جديد في Google Sheet على حسابك، والمرفقات تُحفظ في مجلد Google Drive. **ما يحتاج تعطيني أي باسورد.**

> الوقت المتوقع: ٣–٥ دقائق.

---

## 1) الشيت جاهز ✅
أنشأنا لك الجدول على حسابك:
**[بريفات دولاب · Doolab Briefs](https://docs.google.com/spreadsheets/d/1z6H_ab9W-CTKq_iLd9uVsDdYqU1GhSch3ngT7VQG9Uo/edit)**

افتحه من الرابط (لاحظ إنه فاضي الآن — تبويب «الهوية» يتكوّن تلقائياً مع أول إجابة).

## 2) الصق السكربت
1. من القائمة العلوية: **Extensions ← Apps Script** (إضافات ← Apps Script).
2. احذف أي كود ظاهر، والصق كامل محتوى الملف [`Code.gs`](Code.gs).
3. اضغط أيقونة الحفظ 💾.

## 3) انشر كـ Web App
1. أعلى يمين الصفحة: **Deploy ← New deployment** (نشر ← نشر جديد).
2. اضغط الترس ⚙️ بجانب "Select type" واختر **Web app**.
3. عبّئ:
   - **Description**: `Doolab Briefs`
   - **Execute as**: **Me** (أنا)
   - **Who has access**: **Anyone** (أي شخص) ← مهم جداً عشان يستقبل من الموقع.
4. اضغط **Deploy**.
5. أول مرة يطلب صلاحيات: **Authorize access** ← اختر حسابك ← لو ظهر تحذير اضغط **Advanced ← Go to … (unsafe)** ← **Allow**.
   (هذا طبيعي لأن السكربت يخصّك أنت.)
6. انسخ رابط **Web app URL** (ينتهي بـ `/exec`).

## 4) فعّل الرابط في الموقع
افتح ملف [`../config.js`](../config.js) وحط الرابط:

```js
const ENDPOINT_URL = "https://script.google.com/macros/s/AKfy....../exec";
```

احفظ الملف. خلاص ✅ — جرّب تعبّي الاستبيان، ولاحظ يطلع صف جديد في تبويب **«الهوية»**.

---

## ملاحظات
- **تبويبات منفصلة:** التبويب يُنشأ تلقائياً حسب نوع الاستبيان (`الهوية` / `تسويق` / `تقنية`). لما نضيف استبيان التسويق لاحقاً، يروح لتبويبه بنفس الملف.
- **المرفقات:** الصور والملفات تنحفظ في مجلد Drive اسمه «Doolab Briefs - مرفقات»، ويظهر رابط كل ملف في عمود «المرفقات».
- **تصدير Excel:** من الشيت: `File ← Download ← Microsoft Excel (.xlsx)`.
- **تعديل السكربت لاحقاً:** أي تعديل، لازم تعيد **Deploy ← Manage deployments ← Edit ✏️ ← Version: New version ← Deploy** عشان يطبّق التغيير (نفس الرابط يبقى).
- **اختبار سريع:** افتح رابط الـ `/exec` في المتصفّح، المفروض يطلع: `Doolab Briefs endpoint is live ✦`.
